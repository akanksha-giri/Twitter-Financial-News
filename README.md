# Twitter Financial News Classification

## Overview
A multi-class text classification project that categorizes finance-related tweets into 20 financial news categories such as Earnings, Markets, Fed/Central Banks, IPO, and Stock Movement.

## Dataset
- Twitter Financial News Dataset  
- Training data: 16,990 tweets  
- Validation data: 4,117 tweets  
- 20 financial news categories  

## Approach
- Text preprocessing and cleaning  
- TF-IDF feature extraction  
- Models used:
  - Logistic Regression  
  - Linear SVM  
  - Artificial Neural Network (ANN)

## Evaluation
- Accuracy, Precision, Recall, Macro F1-score  
- Confusion matrices for all models  

## Tools
Python, Pandas, Scikit-learn, Matplotlib, Seaborn, WordCloud, Jupyter Notebook
